# 프로젝트 개요 (Project Overview)

## 1. 프로젝트 목표
*   `mentalhealth.go.kr` 사이트의 메인 페이지 (`index.html`)를 **새로운 환경에서 처음부터 직접 구현**하는 것을 목표로 합니다.

## 2. 학습 목표 및 접근 방식
*   **단순 복사-붙여넣기 지양:** 기존 코드를 단순히 복사하여 붙여넣는 방식이 아닌, 사용자님께서 스스로 HTML, CSS, JavaScript 코드를 작성하며 학습하고 웹 개발 능력을 향상시키는 것을 최우선 목표로 합니다.
*   **'시각적 설계도'로서의 참고 자료:** 기존 `mental_health_portfolio/index-1.html` 또는 실제 `mentalhealth.go.kr` 웹사이트를 '시각적 설계도'이자 '기능 명세서'로 활용합니다. 즉, 브라우저에서 참고 자료를 보면서 페이지의 '모습'과 '기능'을 파악하되, **그 코드 자체를 직접 복사하지 않고** 사용자님의 새로운 프로젝트에서 코드를 직접 작성합니다.
*   **새로운 환경에서의 시작:** 기존 프로젝트(`projectA`)와 분리된 완전히 새로운 디렉토리에서 프로젝트를 시작하여, 깨끗하고 명확한 학습 환경을 조성합니다.

## 3. 핵심 개발 원칙
*   **시맨틱 HTML:** 문서의 의미를 명확히 하는 HTML5 태그를 적극적으로 활용하여 웹 표준 및 웹 접근성을 준수합니다.
*   **모듈화된 CSS/JS:** 파일들을 목적에 맞게 분리하고, 재사용성을 고려하여 깔끔하고 유지보수하기 쉬운 코드를 작성합니다.
*   **순수 JavaScript 우선:** 가능한 한 순수 JavaScript(Vanilla JS)를 사용하여 웹의 동작 원리를 깊이 이해하고 핵심 역량을 강화합니다.
